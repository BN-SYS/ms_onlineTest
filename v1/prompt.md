다음 요구사항 분석 내용을 기반으로 멘사 온라인테스트를 HTML, CSS, JavaScript로 실제 동작하는 프로토타입 만들어줘.
파일 구조는 동일한 코드를 여러 파일에 적지 않고, 유지보수 가능하도록 모듈화 해줘. (공통소스 및 필요시 각 화면별 전용으로 분리)

---

# 1. 프로젝트 목적

### 1) 기획 배경

- 기존 **입회 테스트 외 별도 온라인 테스트 콘텐츠 신설**
- 누구나 체험 가능 (회원/비회원)
- 무료로 응시 가능
- **결과 확인 시 결제 필요**

### 2) 목표

- 멘사 브랜드 기반 IQ 체험형 콘텐츠 제공
- 신규 유입 확보 및 데이터 축적
- 결과 열람 유료화로 수익 모델 확보

---

# 2. 서비스 구조 (사용자 플로우)

### ① 접근

- 회원 / 비회원 모두 이용 가능

---

### ② 테스트 진행

- 문제 및 정답: 관리자 수정 가능
- 보기 순서: **랜덤 노출**

---

### ③ 결과 처리

- 결제 전 사용자 정보 입력
    
    **입력 항목**
    
    - 이름
    - 출생연도 (채점 로직 활용)
    - 이메일
    - (회원일 경우 자동 ID 매핑)
    - 약관 동의
    - IP 자동 저장

| 구분 | 내용 |
| --- | --- |
| 응시 완료 | 점수 산출 |
| 기본 안내 | 결제 필요 안내 |
| 결제 전 | 상세 결과 비공개 |
| 결제 후 | IQ 추정 결과 및 상세 결과 열람 가능 |

---

# 3. 채점 및 로직

### 1) 채점 기준

- 정답 수 기반 점수 산출
- **출생연도(나이) 고려 보정 로직 적용**
    - 연령대별 환산 점수 적용
    - 대표님 별도 로직 전달 예정

---

# 4. 결제 시스템

- PG사: **이니시스**
- 멘사 별도 상점 ID 생성 중
- 결제 완료 후 결과 열람 권한 부여
- 결제 내역 관리자에서 조회 가능

---

# 5. 관리자 기능 정의

## 5-1. 문제 관리

| 항목 | 기능 |
| --- | --- |
| 문제 수정 | 가능 |
| 문제 이미지 | 통 이미지 1개 업로드 |
| 보기 | 개별 이미지 업로드 |
| 정답 설정 | 관리자 지정 |
| 보기 순서 | 자동 랜덤 |
| 보기 개수 | 최대 8개 |

**특이사항**

- 8개 미만 입력 시
    
    → 나머지 보기는 자동 빈칸 처리
    

---

## 5-2. 응시 로그 관리

결제 여부와 관계없이 아래 정보 저장

| 항목 | 저장 여부 |
| --- | --- |
| IP | 저장 |
| 회원 ID (로그인 시) | 저장 |
| 이름 | 저장 |
| 출생연도 | 저장 |
| 이메일 | 저장 |
| 응시 시간 | 저장 |
| 점수 | 저장 |
| 결제 여부 | 저장 |
| 결제 내역 | 저장 |

→ 관리자에서 조회 가능

---

# 6. 회원/비회원 정책

| 구분 | 가능 여부 |
| --- | --- |
| 회원 | 가능 |
| 비회원 | 가능 |
| 비회원 결과 열람 | 결제 후 가능 |
| 회원 결과 열람 | 결제 후 가능 |

※ 회원 여부는 단순 로그 구분 목적

---

# 7. 시스템 요구사항 정리

### 1) 필수 기능

- 문제 랜덤 보기 처리
- 연령 보정 채점
- PG 연동
- 로그 저장
- 관리자 문제 수정 기능

### 2) 보안 고려

- IP 중복 응시 체크 여부 협의 필요
- 동일 이메일 반복 결제 정책 협의 필요
- 결과 URL 직접 접근 차단

---

# 8. MVP 범위 제안

### 포함

- 문제 관리
- 랜덤 보기
- 연령 보정 채점
- PG 결제 후 결과 열람
- 관리자 로그 조회

### 추후 고도화 가능

- 통계 대시보드
- 평균 점수 분석
- 연령대별 분포 분석
- 재응시 제한 정책
- 결과 PDF 자동 생성

---

# 9. 확인 필요 사항 (모호한 지점)

1. 동일 IP 재응시 제한 여부
2. 동일 이메일 재결제 허용 여부
3. 결제 취소 시 결과 접근 차단 여부
4. 결과 노출 범위 (점수만 / 해설 포함 여부)
5. 결과 저장 기간

---

# 10. 한 줄 요약

> 무료 체험 기반 IQ 테스트를 제공하고, 결과 열람을 유료화하며,
> 관리자에서 문제 수정·로그 추적·결제 관리가 가능한 구조의 온라인 테스트 시스템 구축.
